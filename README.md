# Telegram to VK notifier

Desktop-приложение для Windows, которое работает как Telegram userbot, живет в системном трее и отправляет уведомления в личные сообщения VK при получении нового личного сообщения от Telegram-контакта.

## Что есть сейчас

- desktop UI на `PyQt5`
- запуск без консоли через отдельное окно
- иконка в системном трее
- авторизация Telegram по QR
- авторизация Telegram по номеру телефона, коду и 2FA-паролю
- запуск и остановка мониторинга из интерфейса
- опциональное удаление уведомлений в VK после прочтения сообщений в Telegram
- логирование в окно приложения и в `%AppData%\TgVkNotifier\logs\app.log`
- сохранение настроек в `%AppData%\TgVkNotifier\app_config.json`
- хранение Telegram-сессии в `%AppData%\TgVkNotifier\session\`

## Установка

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Где взять данные

### Telegram API

1. Перейдите на `https://my.telegram.org`.
2. Откройте `API development tools`.
3. Создайте приложение.
4. Скопируйте `api_id` и `api_hash`.

### VK token

1. Откройте нужное сообщество VK.
2. Перейдите в `Управление` -> `Работа с API` -> `Ключи доступа`.
3. Создайте ключ с правом `messages`.
4. Включите `Сообщения сообщества`.
5. Напишите этому сообществу с личного аккаунта хотя бы одно сообщение.

## Запуск desktop-приложения

```bash
python desktop_app.py
```

## Как пользоваться

1. Откройте вкладку `Настройки`.
2. Заполните `TG API ID`, `TG API Hash`, `VK Token`, `VK User ID`.
3. Нажмите `Сохранить настройки`.
4. Перейдите во вкладку `Авторизация`.
5. Выполните вход одним из способов:

### Вход по QR

1. Нажмите `Показать QR`.
2. В Telegram откройте `Настройки` -> `Устройства` -> `Подключить устройство`.
3. Отсканируйте QR-код.
4. Если включена двухэтапная защита, введите пароль 2FA в приложении.

### Вход по номеру телефона

1. Введите номер телефона в международном формате.
2. Нажмите `Получить код`.
3. Введите код из Telegram.
4. Если включена двухэтапная защита, введите пароль 2FA.

### Запуск мониторинга

1. После успешной авторизации нажмите `Старт`.
2. Приложение начнет слушать новые личные сообщения.
3. Если включен чекбокс `Удалять уведомления в VK после прочтения в Telegram`, уведомления будут удаляться в VK после чтения диалога в Telegram.
4. При закрытии окна оно не завершится, а свернется в трей.
5. Для полного выхода используйте меню иконки в трее.

## Сборка в `.exe`

После установки зависимостей можно собрать приложение:

```bash
build_app.bat
```

Результат появится в папке `dist`.

Если нужно собрать вручную:

```bash
pyinstaller --noconsole --name TgVkNotifier desktop_app.py
```

## Хранение файлов

- настройки приложения: `%AppData%\TgVkNotifier\app_config.json`
- связки сообщений Telegram/VK: `%AppData%\TgVkNotifier\message_links.json`
- лог-файл: `%AppData%\TgVkNotifier\logs\app.log`
- сессия Telegram: `%AppData%\TgVkNotifier\session\tg_userbot_session.session`

## Миграция старых данных

- если рядом с приложением или в папке проекта уже есть старый `app_config.json`, приложение сможет прочитать его при первом запуске
- если рядом с приложением или в папке проекта уже есть старая `*.session`, приложение использует ее как источник для мягкой миграции в `%AppData%`
- старые файлы автоматически не удаляются

## Примечания

- `app_config.json` и `.env` не должны попадать в публичный репозиторий.
- Если VK возвращает ошибку `901`, пользователь еще не разрешил сообщения от сообщества.
- Уведомления отправляются только для личных сообщений от контактов или взаимных контактов Telegram.
- Старый консольный скрипт `tg_to_vk_userbot.py` оставлен в проекте, но основной режим теперь desktop.
